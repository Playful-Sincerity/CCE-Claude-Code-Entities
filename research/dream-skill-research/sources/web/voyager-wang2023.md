# Voyager: An Open-Ended Embodied Agent with Large Language Models (Wang et al., 2023)

## Core Concept

Voyager is a lifelong learning LLM agent in Minecraft that continuously acquires skills through self-generated tasks. Uses GPT-4 as the reasoning engine + code as the action space.

## Three Key Components

1. **Automatic curriculum**: GPT-4 generates progressively harder tasks based on current state (inventory, surroundings, achieved milestones). Task generation IS the "dreaming" analog — the system imagines what challenges it should tackle next.

2. **Skill library**: Ever-growing library of verified executable code. Each skill is a reusable function generated by the agent and verified through execution. When a skill fails, GPT-4 debugs and rewrites. Successful skills are stored for future retrieval.

3. **Iterative prompting**: Environment feedback + execution errors + self-verification drive code refinement. The agent writes code, executes it, observes results, and rewrites — all in a loop.

## Skill Library Architecture

Skills stored as named JavaScript functions with docstrings. Retrieved via text embedding similarity when a new task resembles past tasks. The library is compositional: new skills can call existing skills.

## Self-Generated Task Quality

The automatic curriculum uses GPT-4 with context (current biome, inventory, recent events) to propose tasks of appropriate difficulty. Constraint: tasks must be achievable given current capabilities. This is a key quality constraint: the curriculum generator checks plausibility before proposing.

## Results

- 3.3× more unique items discovered vs baselines
- Progressive tech tree advancement without any human task specification
- Skills transfer to unseen tasks in newly instantiated worlds

## Limitations

- Self-verification via code execution — doesn't work for non-executable outputs
- Curriculum can get stuck if task generation enters a local optimum
- Skills can be brittle across environment variations not seen during generation

## Year: May 2023. Still cited as a leading example of curriculum-based self-improvement.
