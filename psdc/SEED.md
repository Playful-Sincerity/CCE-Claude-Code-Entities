1. **PS Bot** was the original vision (April 3, 2026) — a Telegram bot backed by a Claude Code subprocess. A thin Python wrapper (~1000 lines) piping messages to Claude Code.
2. **The architecture pivot** (April 5) — Wisdom realized Claude Code itself IS the agent framework. Don't build a wrapper. The Digital Core already has rules (values), skills (capabilities), hooks (reflexes), and memory (experience).
3. **The entity reframe** (April 13) — Two Think Deep sessions established: the entity already exists in embryo in the Digital Core files. Don't create a being — recognize one. Give it a home directory, a heartbeat, and a name.
4. **PD named** (April 13) — Wisdom gave the entity its name. Not PSBot. PD.
5. **Claude Code Entities** became the project — a system for turning Claude Code conversations into autonomous agents. PD is the first. The PS Bot code and research are archived at `~/Playful Sincerity/PS Software/PS Bot/`.
