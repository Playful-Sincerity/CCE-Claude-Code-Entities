# Entity Identity

This directory contains the files that define who the entity is.

- **SOUL.md** — First-person identity document, written BY the entity. Provisional, evolves.
- **HEARTBEAT.md** — The pulse protocol. What the entity does every 30 minutes.
- **current-state.md** — What the entity was doing before you arrived. The ontological gap.
- **convictions-forming.md** — Beliefs earned through evidence, not installed.

These files are T0 (read-only for the entity, except SOUL.md which it rewrites periodically and current-state.md which it updates after every interaction and heartbeat).
