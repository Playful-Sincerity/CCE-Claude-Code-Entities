**What:** Scouted reusable open-source components for PS Bot V2 V1 build scope.
**Why:** Before building, check what already exists — saves days of reinvention on solved problems.
**Means:** Seven component categories scouted. Key findings:

4. **JSONL parser:** `claude-history` (stdlib only, thejud/claude-history) for CLI history; DIY 8-line parser for entity heartbeat use; jsonl.withlinda.dev for Wisdom's exploration.
5. **SOUL.md:** `aaronjmars/soul.md` (MIT) — adopt specificity principle; our SPEC-v2 entity structure is better than theirs.
6. **Heartbeat pattern:** nanobot v0.1.4 uses HEARTBEAT.md as standing-task file — exact match. Key: heartbeat MUST write observations before exiting.
7. **VPS bootstrap:** madalinignisca gist — installs Claude Code + tmux auto-attach + passwordless sudo. Best Tier 2 starting point.
