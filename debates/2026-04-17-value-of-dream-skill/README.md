# Debate: Value of the /dream Skill

**Date:** 2026-04-17
**Mode:** Adversarial (3 rounds)
**Models:** Sonnet debaters → Opus synthesis
**Verdict:** CON wins the default case; PRO wins a narrow high-stakes use case. Keep the skill, restrict its triggers, add drift-comparison.
